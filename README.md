# WeatherWizard
여러 개의 날씨 제공사의 예보 정보를 비교할 수 있는 안드로이드 날씨 애플리케이션

## 제공 기능

- 현재 날씨
- 시간별 예보
- 일별 예보
- 예보 정보 비교
   - 대한민국 기상청(Kma)
   - 노르웨이 기상청(Met Norway)
- 기상 레이더
- 대기질
- 일출/일몰 정보
- 상시 알림
  - 현재 날씨, 시간별 예보 통합 제공
- 일일 알림
  - 매일 정한 시각에 날씨 정보 알림
  - 시간별 예보
  - 일별 예보
  - 현재 날씨
  - 현재 대기질
  - 대기질 예보
- 위젯
   - 11 종류의 위젯
